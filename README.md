# d-security

### About

Security system using Intel Edison + Arduino Breakout board + Groove Seeed shield running on NodeJS

### Hardware: 

* LCD dispay
* LED Lights RGB
* Number Keypad
* Alarm/Buzzer
* Motion PIR sensor



###Jobs

* Active Alarm
  - Numberpad
* Motion Set Off
  - Flash Lights
  - Buzzer
  - Notices
* Disarm Alarm
  - Numberpad
* Logging
  - Notifications
* Users
